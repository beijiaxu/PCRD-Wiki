# PCRD-Wiki
## Feature
    Area drops
    Equipment lookup
    Arena search and resolver  (developing)
    
## How to develop
    env: jdk 1.8, node.js > 10.0, yarn
    import maven dependencies
    mvn clean compile
    start PcrdApplication.java
    cd pcrd-ui
    yarn
    yarn run dev
    visit locahost:3000
